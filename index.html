AI 简历优化与面试模拟 Agent

这是放置在 giveme/index.html 的项目入口页，可被静态渲染平台直接打开。完整交互能力由同目录下的 Streamlit 应用 app.py 提供。

查看 Streamlit 应用脚本 查看依赖文件
01 简历解析 支持粘贴文本，或上传 PDF、DOCX、TXT、MD 文件。
02 岗位匹配 基于简历和 JD 输出匹配度、关键词、风险点和优化建议。
03 面试训练 生成定制化问题，并对回答进行结构化评分。

可渲染入口

本页面不依赖构建工具，浏览器直接打开即可渲染，满足项目目录必须包含 index.html 的要求。

演示模式

Streamlit 应用内置示例简历和 JD，可在无 API Key 时展示页面主流程。

真实调用

需要在侧边栏输入个人 OpenAI API Key，应用不会使用服务端环境变量作为 fallback。

本地运行命令

  1. 进入项目目录：cd giveme
  2. 安装依赖：python -m pip install -r requirements.txt
  3. 启动应用：python -m streamlit run app.py

安全状态

已加固：模型白名单、上传大小限制、DOCX ZIP 元数据校验、用户侧错误脱敏、AI 空响应失败处理。
